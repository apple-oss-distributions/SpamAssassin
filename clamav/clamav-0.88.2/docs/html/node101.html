ClamAV module for ProFTPD

Homepage: http://www.uglyboxindustries.com/open-source.php
Supports: libclamav
This is an add on module for ProFTPD that enables the FTP server to scan newly uploaded files for viruses, before allowing the file upload to complete. This is very valuable, as you, the administrator, can ensure that your FTP sites do not contain viruses uploaded by your users.
