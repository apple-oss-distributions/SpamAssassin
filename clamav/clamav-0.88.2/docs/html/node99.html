HTTP Anti Virus Proxy

Homepage: http://www.server-side.de/
Supports: libclamav
HAVP is a proxy with an antivirus filter. It does not cache or filter content. At the moment the complete traffic is scanned. A reason for that is the chance of malicious code in nearly every filetypes e.g. HTML (JavaScript) or JPEG files.
